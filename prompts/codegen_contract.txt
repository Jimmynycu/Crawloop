The crawler protocol (from crawloop.contract):

    class Crawler(Protocol):
        family: str        # stable id for this page family, e.g. "host/product_list"
        schema_ref: str    # e.g. "Product@1"
        async def crawl(self, url: str, ctx: FetchContext) -> CrawlResult: ...

    class CrawlResult(BaseModel):
        items: list[dict]          # one dict per record; each must match the schema
        next_url: str | None = None  # absolute URL of the next page, or None

The injected context you receive as `ctx` (FetchContext) — your ONLY I/O:

    async def fetch(self, url: str) -> str
        # returns the page HTML; enforces allowlist + rate limit + access recovery
    async def fetch_rendered(self, url: str, wait_for: str | None = None) -> str
        # same, but via a real browser for JS-rendered pages
    def absolutize(self, base: str, href: str | None) -> str | None
        # urljoin(base, href); returns None for a falsy href
    def parse_money(self, raw: str | None) -> Decimal | None
        # pulls the first numeric value out of a price string, e.g. "£51.77" -> 51.77
    def clean_text(self, raw: str | None) -> str | None
        # collapses runs of whitespace and strips the ends

Construct items as plain dicts. Use parsel's Selector(text=html) to query the
DOM. Make every link absolute with ctx.absolutize(url, href) before putting it in
an item or in next_url.
